Комментарии к версии #1

"Формально" задание выполнено, но:
1. Функция change_dir() "номинальная" (нефункционирующая), т.к. в рамках концепции запуска скрипта из командной строки
с параметрами файловый менеджер заново запускается для каждого действия, результат смены директории не сохраняется
для следующей операции.

2. Побочной ошибкой предыдущего пункта является то, что запись в лог окончания операции смены директории происходит
не в тот лог, в который было записано начало этой операции - из-за открытия файл на дозапись в текущей рабочей
директории (cwd). При отсутствии в "сменённой" директории файла лога создаётся новый.

Эти недочёты легли в основу создания следующей версии консольного файлового менеджера (см. v2)
